# VISTA-RDC 🇨🇩
> **L'Excellence du Voyage au Cœur de l'Afrique.**

VISTA-RDC est une plateforme touristique nationale de nouvelle génération conçue pour transformer l'expérience de voyage en République Démocratique du Congo. Alliant technologie moderne (IA, Fintech) et design premium.

## 🌟 Fonctionnalités Clés
- **Interface Premium** : Design sombre avec animations fluides.
- **Gestion par IA** : Assistant intelligent et marketing autonome intégrés.
- **Paiements Hybrides** : Support Mobile Money (M-Pesa, Orange) et Cartes.
- **Tableau de Bord IPO** : Suivi de la valorisation pour entrée en bourse.

## 🚀 Technologies
- React 19, TypeScript, Tailwind CSS 4.0, Supabase.
